# Article Popularity Prediction

## Overview
This repository contains the code and report for a machine learning project on predicting online news article popularity (measured via shares).  
We evaluate whether unconventional composite indices (originality and effort) add predictive power, and compare them against standard structural, topical, timing, and sentiment-based features.
